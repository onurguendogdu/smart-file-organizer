# Changelog

All notable changes to this project are documented in this file.

## [1.2.0] - 2026-07-10

### Added

- Optional processing of files from subfolders
- Persistent storage of the last selected folder
- Persistent storage of the window size
- Persistent storage of the subfolder option
- Button for opening the latest log file
- Detailed processing duration
- Improved result statistics
- Version information in the application window
- Improved validation and error handling

### Changed

- Improved graphical user interface
- Improved progress information
- Improved log file format
- Improved file collection and category handling
- Category folders and log folders are excluded from recursive processing
- Duplicate files are renamed instead of overwritten

## [1.1.0] - 2026-07-10

### Added

- Detailed log files
- Category statistics
- Progress tracking
- Additional file categories

## [1.0.0] - 2026-07-09

### Added

- Initial graphical user interface
- Automatic file categorization
- Automatic folder creation
- Duplicate file protection
- Support for common file extensions
